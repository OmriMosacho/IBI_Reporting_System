AnalyticsDashboard

Displays analytical visualizations for different data domains (Customers, Companies, Stocks). Users can select which subject to analyze via a dropdown. Each subject shows corresponding charts. The dashboard fetches aggregated analytics data from backend endpoints using a shared API client.
